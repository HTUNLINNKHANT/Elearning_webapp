## Image Guidelines

### Style
- Clean, professional, modern aesthetic
- Blue and white color theme
- Japanese cultural elements where appropriate
- High quality, well-lit photos

### Technical Requirements
- Format: JPG or PNG
- Optimization: Compress for web (use tools like TinyPNG)
- Responsive: Provide high-resolution images that scale well

### Placeholder Solution
Until actual images are added, the website uses:
- Gradient backgrounds with text overlays for courses
- Colored circles with initials for student photos
- Gradient boxes with titles for gallery items

## Adding Images

1. Place your images in this directory (`frontend/public/images/`)
2. Use the exact filenames listed above
3. Optimize images before uploading
4. Test on different screen sizes

## Alternative: Using Unsplash or Stock Photos

You can use free stock photos from:
- [Unsplash](https://unsplash.com/) - Search for "japanese classroom", "students learning", etc.
- [Pexels](https://www.pexels.com/)

Make sure to check licensing requirements for commercial use.
